what is Grid?

CSS grid layout is a two-dimensional layout system for the web. It lets you organize content into rows and columns and offers many features to simplify the creation of complex layouts.

Why we use grid instead of flex?

In flex we only align item In row or column, but in grid I can run both in same time.

Grid Properties

  1. Display grid- activate the layout
  2. column-gap for gaps between columns
  3. row-gap for gaps between rows
  4. gap as a shorthand for both
  5. grid-column shorthand for grid-column-start and grid-column-end
  6. grid-row shorthand for grid-row-start and grid-row-end
