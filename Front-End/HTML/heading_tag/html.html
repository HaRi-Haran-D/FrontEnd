Header Element

Represents introductory content, often containing headings, logos, or navigation.

Navigation Element

Defines a set of navigation links.

Article Element

It specifies independent, self-contained content.

Section Element

It defines a section in a document.

Main Element

Represents the primary content of a document, used only once per page to outline the core information.

Aside Element

Contains content that is tangentially related to the surrounding content, like a sidebar.

Section Element

Groups related content together within a document.

Footer Element

Contains footer information, like copyright details or contact information.
