When the image is located in same folder

Just type the image's file in the Source attribute.


when the image is located in Sub folder

Now you need to image"s folder name and file name in the source attribute.

when the image is located in parent folder

Now then you need to type 2 dots in source attribute (.. means relative reference it helps to move up one level from your current folder) then type file name.

when the image is located in parent folder's sub folder

now then again you need to type 2 dots in source attribute (.. means relative reference) then type the sub folder name and file name in source attribute.
